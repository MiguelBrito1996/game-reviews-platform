# game-reviews-platform
Uma plataforma web onde utilizadores podem criar e partilhar reviews de jogos, conectar com outros reviewers e explorar opiniões sobre os seus jogos favoritos

## Sobre o Projeto
Este é o meu primeiro projeto fullstack, criado para aprender desenvolvimento web. O projeto está em desenvolvimento ativo.

## Objetivos do Projeto
- Aprender desenvolvimento backend (APIs, autentificação, base de dados)
- Dominar frontend moderno (HTML/CSS/JS, frameworks)
- Implementar boas práticas de project management
- Criar um portfólio profissional

## Funcionalidades Planeadas

### MVP (Versão Inicial)

- [ ] Sistema de registos e login
- [ ] Criar e publicar reviews de jogos
- [ ] Ver reviews de outros utilizadores
- [ ] Sistema de ratings (1-5 estrelas ou 1-10)
- [ ] Perfil de utilizador básico

### Futuras expansões
- Reviews de outros tipos de arte (filmes, livros, música)
- Sistema de follows/seguidores
- Comentários em reviews
- Feed personalizado
- Sistema de likes/upvotes
